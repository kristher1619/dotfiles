Coda Plugin and Tips


Coda is a one-window web development application from Panic: http://panic.com/coda/


Plugin


Go to the PeepOpen Preferences menu and choose Coda as your editor. Click the Install Plugin button and you’re all set to use PeepOpen from Coda.




Usage


  • Open a Site in Coda. Without an active Site, PeepOpen won’t work. 
  • Hit Control-Command-Option-T to launch PeepOpen. (If someone can think of a better keyboard shortcut, we’d love to hear it.)
  • Search for a file and hit Enter to open the file in Coda.



Tips


Graphic files are currently ignored by PeepOpen, but future releases may be able to open other kinds of files in Coda.



Uninstall


  • Use the Finder to navigate to ~/Library/Application Support/Coda/Plug-ins
  • Remove PeepOpen.codaplugin and restart Coda.
