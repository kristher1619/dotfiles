MacVim Plugin and Tips


MacVim is a powerful text editor: https://github.com/b4winckler/macvim


NOTE: MacVim Snapshot 55 or better is recommended. Many bugs were fixed specifically for the benefit of PeepOpen users!


    Download: MacVim-snapshot-56.tbz


Plugin


Go to the PeepOpen Preferences menu and choose MacVim as your editor. Click the Install Plugin button and restart MacVim. You’re all set to use PeepOpen from MacVim.



The plugin will be installed under ~/.vim/bundle/ if you use Tim Pope's pathogen meta-plugin. Otherwise, it will go to the standard ~/.vim/plugin directory.


You should also change your MacVim preferences to “Open files from applications in the current window and set the arglist.” Without this, files will spawn separate windows every time.




Usage


Change to a directory, 


:cd ~/src/my_project


Edit at least one file.


:e app.rb


Hit <Leader>-p to activate PeepOpen (which is probably “\-p”).


Search for a file with PeepOpen and hit Enter to return to MacVim.



Tips


See ~/.vim/plugin/peepopen.vim for instructions on changing the hot key to ⌘-T.



Uninstall


Remove ~/.vim/plugin/peepopen.vim or ~/.vim/bundle/vim-peepopen and restart MacVim.
