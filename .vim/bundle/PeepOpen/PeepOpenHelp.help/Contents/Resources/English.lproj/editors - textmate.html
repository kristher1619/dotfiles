TextMate Plugin


TextMate is a full-featured text editor with support for snippets, commands, and syntax highlighting: http://macromates.com/


Plugin


Go to the PeepOpen Preferences menu and choose TextMate as your editor. Click the Install Plugin button, restart TextMate, and you’re all set to use PeepOpen from TextMate.



It's now a native plugin, so it will work even if you don't have a file open (however, you do need to have a TextMate project open).


Usage


  • Open your project in TextMate. 
  • Hit ⌘-T as usual, but PeepOpen will launch and you can search for a file. 
  • Hit Enter to open the file in TextMate.


Uninstall


To remove the plugin completely:


  • Go to ~/Library/Application Support/TextMate/Plugins
  • Remove the PeepOpen plugin
  • Restart TextMate
