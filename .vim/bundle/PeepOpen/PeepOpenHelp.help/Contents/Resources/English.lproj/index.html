PeepOpen


PeepOpen is an editor-independent fast file switcher.


It was developed for computer programmers, but is useful for anyone who needs to switch between files with style and ease.



Quick Start


See a movie of basic usage: http://peepcode.com/system/uploads/2010/peepopen-demo.mov


  • Open the Preferences menu. Choose your text editor and install the PeepOpen plugin for that editor with the Install Plugin button.
  • Open a project in your text editor. In TextMate, you can drag a directory to the TextMate icon to open it.
  • Hit the appropriate keyboard shortcut to activate PeepOpen for the current project. In TextMate, it’s ⌘-T
  • In PeepOpen’s search box, type the name of the file you’re looking for.
  • Hit Enter, and the file will be opened back in your text editor.


PeepOpen is now a menu bar app. Quit by selecting the PeepOpen bottle opener icon and choosing Quit PeepOpen.
