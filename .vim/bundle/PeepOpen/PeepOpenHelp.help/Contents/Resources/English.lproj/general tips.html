General Tips


Our goal is that PeepOpen should just work as you expect it to. But understanding how it searches may help you a bit.


Hide the Menu Bar Icon


You can now hide the menu bar icon. PeepOpen will only appear when you activate it from your editor. You can access the menu via the sprocket icon to the right of the search field.


In Advanced Preferences, uncheck the Show in Status Bar checkbox. Restart PeepOpen.


To enable the menu bar icon again, activate PeepOpen from your editor, use the sprocket to show the Preferences menu, and check the box. Restart and you'll see it back in the menu bar.


Customize Directories and Files


Use the Advanced pane of the Preferences window to customize the regular expression that filters directories and files. If the Regex matches, the file or directory will be omitted from the list you see when you search.


Basically, the directory regex is a chance to match against the beginning of a path (relative to the current project) and the file is a chance to match against the end of the path. Both are applied to the same path string.


The project root regex is used for Xcode projects (and maybe others in the future) to find the starting directory for the project.


You must refresh existing projects to see the results (Menu > Reload Files or Command-R).


Search Algorithm Prefers Filenames


Filenames score higher than paths. So you may find it easiest to start by typing part of a filename. If you recently edited a file, it will probably show up at the top.




The general strategy (in order of priority) is:


  • Filename match
  • Longest match
  • Match score (calculated from the position and number of matches)
  • Modified date & time



Toggle Between Last Few Files


After you open a few files, the previous file will be at the top. You can switch between two files without entering any search terms. Just activate PeepOpen and hit Enter.



Refresh


Most programming tasks involve creating files. While looking at a project in PeepOpen, hit Command-R to refresh the file listing. 


You can even do this while a search is in progress and you'll instantly see any new files that match the search query.



ESC to Start Over


Hit Escape to clear the search field and start over. Or, hit Escape again to hide PeepOpen and go back to your text editor without choosing a new file.



Fuzzy Search


You can search for paths as well as filenames. Adding slashes may help to force a directory to match.



Spaces are ignored and can be typed anywhere. Some people find it useful to use a space as a mental replacement for an underscore or slash.
