Xcode Integration


Xcode is Apple's IDE for writing Mac OS X and iOS applications.


Plugin


You'll create a simple Xcode user script.


Go to the Xcode Script menu and choose "Edit User Scripts..."




Create a new Shell Script.



Enter the following text into the shell script to open PeepOpen with Xcode's current file path. Set a keyboard shortcut to activate it by clicking in the Command key column in the table (under the small flower character).



Finally, open PeepOpen's Preferences and set Xcode as the Editor.



Usage


  • Open your project in Xcode. 
  • Then open at least one file in Xcode and put your cursor in it.
  • Hit the command key that you chose above to activate PeepOpen.
  • Hit Enter to open the file in Xcode.


Caveat: Xcode is sensitive. If you've selected a file from the tree view, it won't recognize the project's root and will open a different directory. Be sure to keep your cursor inside an Xcode file before activating PeepOpen.



Uninstall


You can change the keyboard shortcut for PeepOpen, or remove it entirely.


To change the keyboard shortcut:


  • Go to Xcode's User Scripts menu.
  • Choose the Open PeepOpen menu item in the table.
  • Set a different keyboard shortcut by clicking in the Command key column.


To remove the user script completely:


  • Go to Xcode's User Scripts menu.
  • Remove the PeepOpen user script item by clicking it and selecting the - at the bottom.
